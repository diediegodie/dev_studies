# Repositório de Estudos

Este repositório reúne **anotações pessoais, resumos e estudos** relacionados ao universo do desenvolvimento backend.  
O objetivo é servir como um espaço de aprendizado contínuo, organização de ideias e registro de conceitos importantes.

## Observações
- Este é um **repositório pessoal de estudos**: não se trata de documentação oficial ou material didático estruturado.
- Os arquivos estão em formato **Markdown**, para facilitar leitura e edição.
- O conteúdo reflete meu processo de aprendizado e pode conter simplificações ou interpretações próprias.
